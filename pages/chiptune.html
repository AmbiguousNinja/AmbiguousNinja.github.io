Chiptune Music Player

Arduino Project Code

Frequency to Note Header File

In this project I wanted to create a simple circuit that would play "chiptune" songs while displaying the note played (regardless of octave) on a keyboard represented by 12 LEDs.

Instead of using up one of the Arduino's digital I/O pins for each LED, Charlieplexing is used to reduce that number to 4 pins. Below is a schematic of Charlieplexing 4 LEDs and a picture of the built circuit.

The LEDs on the assembled circuit are placed to represent the notes of a single octave on a piano.

The circuit connected to an Arduino Uno. A speaker is attached to Pin 5 and GND.
